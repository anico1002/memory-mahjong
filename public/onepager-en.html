Memory Mahjong

Casual memory game: memorize the tile board and reproduce the sequence in the right order.

Type · Casual / Memory (single screen) Platform · Web mobile-first (Phaser 3) Status · Playable prototype Session · ~2-4 min per game

Concept

A board of mahjong-styled tiles is presented face down. At the start of each level, all tiles flip face up for a few seconds. The player must memorize the positions and, once hidden, reproduce a specific sequence by tapping tiles in the order shown in the objective bar. Each mistake flips the wrong tile face up for 2 seconds and applies a time penalty. The dual challenge: remember where each tile is AND maintain the sequence order under time pressure.

Design Pillars

1. Spatial + Sequential Memory

Players exercise two types of memory simultaneously: remembering each tile's position on the board (spatial) and reproducing them in a specific order (sequential). This combination creates a deeper cognitive challenge than a simple memory match.

2. Progressive Time Pressure

Memorization time decreases level by level (from 5s to 2.5s) while sequences grow longer. The timer creates urgency without being punishing in early levels, but becomes the deciding factor in advanced stages.

3. Informative Failure

When you fail, the tile flips showing what was underneath. Players gain useful information with each mistake, turning failures into learning opportunities within the same run. But the time cost makes abusing this strategy lethal.

4. Instant Visual Readability

No tutorials or explanations needed. Recognizable mahjong tiles, clear icons, visible objective bar. Anyone understands the mechanic in 3 seconds: memorize, remember, tap in order.

Game Flow

3 phases per level: Entry Tiles fly onto the board with cascading animation. Memorize All tiles flip face up for X seconds with visible countdown. Recall Tiles hide. Player taps in the objective sequence order. Timer active.

Power-ups

Key Peek

Temporarily flips a tile of the player's choice for 1 second. Ideal for confirming a doubtful position without risking an error.

Scissors Skip

Skips the current tile in the sequence, marking it as automatically completed. Saves the run when you can't remember a position.

Compass Hint

Briefly highlights the next correct tile with a visual pulse. The player knows where it is, but still needs to tap it.

Flag Freeze

Freezes the timer for 5 seconds. Provides breathing room in high-pressure moments without revealing board information.

Progression & Retention

Star System

1-3 stars per level based on mistakes made and remaining time. 3 stars = 0 mistakes + more than 50% time remaining. Accumulated stars unlock themed worlds.

Combos

Consecutive correct taps build a score multiplier (2x-5x). Visual combo feedback floats above the board. A single mistake resets the multiplier to 1x.

Themed Worlds

Expansion into themed tile sets: animals, food, emojis, flags. Each world with its own aesthetic: background, tile style, effects. Unlocked by accumulating stars.

Long-term Retention

  • Daily challenge: random level with global leaderboard.
  • Streak: playing X consecutive days unlocks power-ups.
  • Collectibles: seasonal limited-edition tile sets.

Future Mechanics

  • Special tiles: bomb (reveals adjacent tiles on hit), wildcard (matches any tile), ice (requires double tap).
  • Time attack mode: endless sequences with limited time. Increasing speed.
  • Duel mode: two players compete to complete the same sequence on mirrored boards.
  • Seasonal events: special themed tiles (Halloween, Christmas) with unique mechanics.

Controls

Mobile (primary): direct tap on board tiles. Haptic feedback on correct taps (short vibration) and errors (double vibration). Tiles flip with simulated 3D animation (horizontal squeeze + lift). Desktop: click on tiles. Same visual feedback.

Aesthetics

  • Tiles: classic mahjong style with illustrated icons. Bottom shadow simulating depth. Mystery back with question mark.
  • Board: textured green background with vignette. Tiles with no visible gaps (edges touching).
  • HUD: top bar with level, timer with progress bar, sequence indicators (pills).
  • Animations: diagonal cascade entry, flip with lift and shadow, shake on error, floating combo text.
  • Typography: Lilita One throughout the game.
